### 독커 빌드
```bash
docker build -t custom-ubuntu .
```



### 독커에 로그인
```bash
docker login
```

### 독커이미지 리스트 검색
```bash
docker images
```

### 이미지 태그 설정
```bash
docker tag <local-image-name>:<tag> <dockerhub-username>/<repository-name>:<tag>
```
